使い方
iwm_HtmlMsgBox.exe [オプション] ...

(例)
iwm_HtmlMsgBox.exe -size=250,160 -title="タイトル" -url="help/index.html" -checkbox="上記内容を承諾します。" -button=1,1 -buttontext="はい","いいえ","閉じる"

オプション (例)
-size=width,height 250,160
-title="" "タイトル"
-url="" "help/index.html"
"file://..." "http://..." "https://..."
-checkbox="" "上記内容を承諾します。"
-button=n,n 1,1 => [はい] [いいえ] [閉じる]
1,0 => [はい] [閉じる]
0,1 => [いいえ] [閉じる]
0,0 => [閉じる]
-buttontext="","","" "はい","いいえ","閉じる"

戻り値 (例)
[はい] 1
[いいえ] 2
[閉じる] 0
